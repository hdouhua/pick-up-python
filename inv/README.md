## 构建稳健的投资理财塔

知道：这个世界上到底有哪些投资标的？

- 固收类投资标的：预期收益固定的投资，包括债券、保险、银行理财
- 股票类投资标的：波动大、收益高、风险也高。包括股票、股票型基金
- 实物类投资标的：投资金额较大、流动性较差。典型的包括房产、大众商品（黄金、原油）、艺术收藏品等。
- 新兴投资标的：数字货币等
